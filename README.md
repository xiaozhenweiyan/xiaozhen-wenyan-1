# 像素领地争夺战

一款像素风格的多人在线领地争夺网页游戏。玩家可以创建或加入房间，在2D方块地图上点击占领领地，与朋友一起竞争！

## 🎮 游戏特色

- **像素风格美术**：复古像素风UI，使用 Press Start 2P 像素字体
- **多人联机**：基于 PeerJS (WebRTC P2P) 实现实时多人联机
- **领地争夺**：在草地方块上点击创建你的领地，已占领方块不可覆盖
- **房间系统**：支持公有房间和私有房间（邀请码+密码）
- **简单上手**：无需注册，输入昵称即可开始游戏

## 🕹️ 游戏玩法

1. **开始游戏** → 输入昵称注册临时账户
2. **创建房间** → 设置房间名称、公有/私有、密码（可选），获得邀请码
3. **加入房间** → 浏览公共房间列表，或输入邀请码加入私有房间
4. **占领领地** → 在地图上点击草地方块创建你的领地
5. **开始游戏** → 房间内≥2人时，房主可点击"开始游戏"

## 🗺️ 地图说明

- **草地**（绿色区域）：可点击创建领地
- **海水**（蓝色区域）：不可占领，环绕地图四周
- **领地**：半透明颜色方块，表示玩家占领的区域

## 🛠️ 技术栈

- **HTML5 Canvas**：2D方块地图渲染
- **CSS3**：像素风格样式、动画效果
- **JavaScript ES6+**：游戏逻辑、状态管理
- **PeerJS**：WebRTC P2P 多人联机
